# Moonwhisk Vale App Store Pages

This repository hosts public static pages for Moonwhisk Vale App Store review, support, privacy, and advertising verification.

## URLs after GitHub Pages is enabled

- Landing page: `https://facta-leopard.github.io/Moonwhisk_Vale/`
- Privacy Policy: `https://facta-leopard.github.io/Moonwhisk_Vale/privacy.html`
- Support: `https://facta-leopard.github.io/Moonwhisk_Vale/support.html`
- Project-page app-ads.txt: `https://facta-leopard.github.io/Moonwhisk_Vale/app-ads.txt`

## AdMob app-ads.txt note

AdMob uses the hostname of the App Store developer website / marketing URL and crawls `app-ads.txt` from that host's root. For the `facta-leopard.github.io` host, the root verification URL is:

`https://facta-leopard.github.io/app-ads.txt`

The root user-site repository `Facta-Leopard.github.io` should keep the same seller line:

```text
google.com, pub-7572108671787552, DIRECT, f08c47fec0942fa0
```
